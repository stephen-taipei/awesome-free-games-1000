# 俄羅斯方塊 Tetris

> Game #002 - 經典方塊消除益智遊戲

## 遊戲說明

俄羅斯方塊是一款經典的方塊消除遊戲。玩家需要操控從畫面上方落下的方塊，透過旋轉和移動來排列它們，當一整行被填滿時會自動消除並得分。

## 操作方式

### 電腦鍵盤
| 按鍵 | 動作 |
|------|------|
| ← / A | 左移 |
| → / D | 右移 |
| ↓ / S | 軟降（加速下落） |
| ↑ / W | 順時針旋轉 |
| Z | 逆時針旋轉 |
| 空白鍵 | 硬降（直接落到底部） |
| C | 暫存方塊 |
| P / Esc | 暫停 |

### 行動裝置
- 使用畫面下方的觸控按鈕進行操作

## 遊戲規則

1. 七種不同形狀的方塊（I, O, T, S, Z, J, L）會依序落下
2. 使用方向鍵移動方塊，使其排列整齊
3. 當一整行被方塊填滿時，該行會消除並得分
4. 同時消除多行可獲得更高分數（Tetris = 4行）
5. 消除行數累積可提升等級，等級越高方塊下落越快
6. 當方塊堆積到頂部時，遊戲結束

## 計分規則

| 消除行數 | 基礎分數 |
|---------|---------|
| 1 行 (Single) | 100 |
| 2 行 (Double) | 300 |
| 3 行 (Triple) | 500 |
| 4 行 (Tetris) | 800 |

實際得分 = 基礎分數 × 當前等級

## 技術實現

- **遊戲引擎**: Canvas 2D API
- **程式語言**: TypeScript
- **方塊系統**: 7-bag 隨機系統
- **旋轉系統**: 支援 Wall Kick 牆踢
- **特色功能**: 影子預覽、暫存方塊、觸控控制
- **多國語言**: 繁中/簡中/英/日/韓
