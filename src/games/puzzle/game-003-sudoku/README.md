# 數獨 Sudoku

> Game #003 - 經典 9×9 數字邏輯推理遊戲

## 遊戲說明

數獨是一款經典的數字邏輯遊戲。玩家需要在 9×9 的網格中填入數字 1-9，使得每一行、每一列、每個 3×3 宮格內的數字都不重複。

## 操作方式

### 電腦鍵盤
| 按鍵 | 動作 |
|------|------|
| 1-9 | 填入數字 |
| ← ↑ → ↓ | 移動選擇 |
| Backspace / Delete | 清除格子 |
| N | 切換筆記模式 |
| H | 使用提示 |

### 滑鼠/觸控
- 點擊格子選擇
- 使用下方數字鍵盤填入數字
- 點擊「筆記」切換筆記模式
- 點擊「提示」獲取幫助

## 遊戲規則

1. 在每個空格填入 1-9 的數字
2. 每一**行**內的數字不能重複
3. 每一**列**內的數字不能重複
4. 每個 **3×3 宮格**內的數字不能重複
5. 填錯 3 次遊戲結束
6. 有 3 次提示機會

## 難度說明

| 難度 | 空格數 | 說明 |
|------|--------|------|
| 簡單 | 35-40 | 適合初學者 |
| 中等 | 41-48 | 需要一些邏輯推理 |
| 困難 | 49-54 | 需要進階技巧 |
| 專家 | 55-60 | 極具挑戰性 |

## 技術實現

- **遊戲引擎**: Vanilla TypeScript + CSS Grid
- **謎題生成**: 回溯法生成有效解，再移除數字
- **驗證系統**: 即時錯誤檢查
- **特色功能**: 筆記模式、提示系統、相關格子高亮
- **多國語言**: 繁中/簡中/英/日/韓
- **深色模式**: 自動跟隨系統設定
