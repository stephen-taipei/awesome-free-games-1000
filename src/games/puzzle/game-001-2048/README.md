# 2048

> Game #001 - 經典數字合併益智遊戲

## 遊戲說明

2048 是一款經典的滑動方塊益智遊戲。玩家需要透過滑動來移動方塊，當兩個相同數字的方塊碰撞時會合併成一個更大的數字。目標是合併出 2048 方塊！

## 操作方式

### 電腦
- **方向鍵**: ↑ ↓ ← → 移動方塊
- **WASD**: W/A/S/D 也可以控制移動

### 行動裝置
- **滑動**: 在遊戲區域上下左右滑動來移動方塊

## 遊戲規則

1. 每次移動，所有方塊會朝該方向滑動到底
2. 相同數字的方塊碰撞時會合併（例如：2+2=4, 4+4=8）
3. 每次移動後會隨機出現一個新方塊（2 或 4）
4. 當方塊填滿且無法再移動時，遊戲結束
5. 成功合併出 2048 方塊即為獲勝！

## 技術實現

- **遊戲引擎**: Vanilla JavaScript + CSS Grid
- **動畫效果**: CSS Transitions + Keyframes
- **響應式設計**: CSS Variables + Media Queries
- **多國語言**: 自訂 i18n 模組（繁中/簡中/英/日/韓）
- **數據追蹤**: Google Analytics 4 整合
- **本地存檔**: LocalStorage 儲存最高分
