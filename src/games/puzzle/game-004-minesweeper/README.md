# 掃雷 Minesweeper

> Game #004 - 經典邏輯推理益智遊戲

## 遊戲說明

掃雷是一款經典的邏輯推理遊戲。玩家需要根據數字提示推理出地雷位置，並揭開所有非地雷格子來獲勝。

## 操作方式

### 電腦滑鼠
| 操作 | 動作 |
|------|------|
| 左鍵點擊 | 揭開格子 |
| 右鍵點擊 | 標記/取消旗標 |
| 中鍵/雙擊 | 快速揭開（Chord） |

### 行動裝置
- **點擊**: 揭開格子
- **長按**: 標記旗標

## 遊戲規則

1. 點擊任意格子開始遊戲（首次點擊保證安全）
2. 數字表示該格子周圍 8 格中有幾顆地雷
3. 使用右鍵標記你認為是地雷的格子
4. 點擊到地雷即遊戲結束
5. 揭開所有非地雷格子即可獲勝

## 難度說明

| 難度 | 網格大小 | 地雷數量 |
|------|---------|---------|
| 簡單 | 9×9 | 10 |
| 中等 | 16×16 | 40 |
| 困難 | 16×30 | 99 |

## 技術實現

- **遊戲引擎**: Vanilla TypeScript + DOM
- **地雷生成**: 首次點擊後隨機分布（排除點擊區域）
- **連鎖展開**: 遞迴揭開空白區域
- **經典風格**: 還原 Windows 95 掃雷視覺風格
- **觸控支援**: 長按標記旗標
- **多國語言**: 繁中/簡中/英/日/韓
